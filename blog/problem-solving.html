Coping with coding block

Coming into the third/fourth week of school, I was feeling pretty good. I'd become quite comfortable with using HTML & CSS in my first two weeks and I figured with the way things were going, I was bound to succeed in this course. Never in my life have I been humbled so fast.

It's only been two weeks of me learning Javascript and somehow, I feel like I'm already somehow falling behind. This past week was especially hard for me. It's the weekend now & I'm trying my best not to stress out about school, but I'm still doing my blog because I'm playing catch up. It's been quite difficult - I've been feeling very confused, overwhelmed and exhuasted. I know this entry so far is sounding quite bleak and pessimistic but there is always a good side.

Though things've been challenging - I believe it may only feel so drastic because I've been so comfortable for the past few years. I've spent the past few years sticking to what I know, staying strictly with-in my comfort zone.

In the four short weeks I've been at Devacademy, I've already experienced so much valuable upskilling. I've been able to use my brain properly for the first time - my brain actually hurts from how much I've been using it and it's great! I'm learning how to regulate my emotions. I'm learning to be kinder to myself & to have more self compassion.

I may've started this course with the idea of a better future in mind - I didn't realize how fun & fulfilling journey there could be too.


Trying to Troubleshoot

"We're going to write a function that takes a number as an argument, if that number is a multiple of 3 the function will return 'Flip'. For multiples of 5 we will return 'Flop'. For numbers that are multiples of both 3 and 5, we will return 'FlipFlop'. If none of these cases are true, we're just going to return the number."

This exercize in particular, I really struggled with. I tried to simplify the problem, research, try different methods - it seemed like nothing I did brought me closer to the answer. The longer I'd spent going around in circles, trying to figure this problem out - the more frustrated & upset I was becoming. I was stuck in a negative feedback loop.

I ended up realizing, I'd get nowhere if I continued this way. It was clear to me, if I wanted to move forwards, I'd need to change something - I'd need to reset. I tunneled my way out of my study zone; went for a walk down the road, grabbed some coffee, came home & took a shower. Put on a non-school-related youtube video while I cooked lasagna. Only after all this, did I sit back down & attempt the problem again.

Funnily enough, even after all of this, I still couldn't figure it out. I ended up reaching out to one of my class-mates & they agreed to help me. We got on call, they guided me through the problem & helped me understand where I was going wrong. They explained the problem in such a way that it was so easy for me to understand, I regretted not reaching out sooner. I've since made it a regular practice to interupt myself in studying/working & to ask for help when I'm stuck!!
